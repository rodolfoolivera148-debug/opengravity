# Agregador de Feeds

Este proyecto es un agregador de feeds que permite recopilar y mostrar contenido de diferentes fuentes.

## Inicio

1. Instalar dependencias: `npm install`
2. Iniciar servidor: `node index.js`

## Uso

1. Abrir navegador y acceder a `http://localhost:3000`
2. Seleccionar fuente de feed y agregar a la lista

## Funcionalidades

* Agregar fuentes de feeds
* Mostrar contenido de las fuentes agregadas
* Filtrar contenido por categorías

## Contribuir

1. Clonar repositorio: `git clone https://github.com/Opengravity/proyecto5-agregador-feeds.git`
2. Crear rama para realizar cambios: `git branch feature/nueva-funcionalidad`
3. Realizar cambios y commit: `git add . && git commit -m "Mensaje de commit"`
4. Subir cambios a repositorio: `git push origin feature/nueva-funcionalidad`
